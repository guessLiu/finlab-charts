# 熱門股週報

## 這是什麼表？特色在哪？

熱門股週報是每週一次的台股資金流向掃描報告。它把本週每個交易日的量能熱點資料彙整起來，找出「在台股科技製造族群中，哪些族群和個股持續吸引資金」。

一般成交量排行榜只反映單日現象，一天的爆量可能只是軋空、消息面等偶發事件。週報的核心特色是**累計性**：一個族群必須在本週多個交易日都符合篩選條件，才會出現在排行榜上。上榜越多天，代表資金持續流入的可信度越高。

### 主要特色

- **族群維度**：以產業族群為單位做排行，而非個股排行，方便判斷資金流向的方向
- **持續性過濾**：出現天數、核心個股等指標都強調跨日的連貫性
- **多維評分**：篩選條件同時考量量能熱度（相對歷史）、量能加速（相對自身近期）、短期漲幅三個維度
- **週訊號摘要**：自動歸納本週主流、新興族群、熱度消退族群等關鍵訊息

## 如何解讀

### 第一步：看「本週訊號」

週報最上方的訊號卡片是全週的摘要結論，快速瞭解這週的市場風格。

| 訊號 | 意思 |
|------|------|
| 🏆 本週主流 | 綜合評分最高的族群，本週資金集中的主要方向 |
| 🔥 全週強勢 | 每個交易日都出現在熱門榜的族群，連續性最強 |
| 🌱 週末新興 | 只在週三之後才出現的族群，可能是新崛起的題材，觀察能否延續 |
| 📉 熱度消退 | 只在週初出現、後來消失的族群，資金已轉向 |
| ⚡ 超高熱度 | 平均熱度百分位達 92% 以上，量能極度集中，警覺是否過熱 |
| 💎 核心個股 | 在某族群中出現最頻繁的個股，族群行情的主力標的 |

**重點判斷**：全週強勢 + 本週主流同為一個族群，代表最強方向。只出現在週末新興的族群，可信度還不夠，下週繼續觀察。

---

### 第二步：看「族群排行」

族群排行表是本週所有出現過的族群，依綜合評分由高到低排列。

| 欄位 | 怎麼看 |
|------|--------|
| 出現日 | 週一到週五各格若有數字，代表當天上榜的個股數量；空白代表當天未上榜 |
| 持續度 | 出現天數 / 本週已統計天數；數字越高、進度條越長，代表越穩定 |
| 平均熱度 | 該族群上榜個股的平均量能熱度百分位，反映量能集中程度 |
| 平均檔數 | 每次上榜時，平均有幾檔個股同步達標，數字越高代表族群性越強 |
| 趨勢 | ▲ 代表週末比週初上榜檔數增加，量能在擴散；▼ 代表縮減 |
| 核心股 | 本週連續上榜 ≥2 天的個股數量 |

**重點判斷**：族群排名高、持續度高、平均檔數多、趨勢向上，四項同時成立才是真正的強勢族群。

---

### 第三步：看「選股明細」

點擊族群排行的任一列，右欄個股索引會自動定位到該族群，展開該族群的所有個股。明細表則列出本週所有上過榜的個股，供進一步篩選。

| 欄位 | 怎麼看 |
|------|--------|
| ★ 核心個股 | 本週上榜 ≥2 天的個股，紅星標記，優先關注 |
| 上榜天數 | 越多天代表持續性越強 |
| 放量位置 | 進度條顯示現價相對近3年高點的位置；高位放量可能是突破，但也需留意風險 |
| 月內位置 | 進度條顯示現價在近1個月高低區間的位置；月底高位代表近期強勢 |
| 加速趨勢 | 折線圖顯示近5個連續時間窗口的量能加速度；持續在1以上代表量能仍在擴張 |
| 1日% / 5日% / 20日% | 短中期漲跌幅，確認漲勢是否仍在持續 |

**重點判斷**：上榜天數多、月內位置高（近期強勢）、加速趨勢線維持在1以上、5日正報酬，四項同時成立的個股是重點追蹤標的。

---

## 特別功能

- **點擊族群排行的任一列** → 右欄個股索引會自動捲動到該族群並展開，不需要手動在右欄尋找
- **選股明細表的「選股」欄排序**：點第一下是核心股（★）優先排列，再點一下才切換為代號字母順序，行為與其他欄不同

## 名詞解釋

**量能熱度百分位**
今天的成交週轉率（成交金額 ÷ 市值），相對該股過去一年歷史的高低位置。數值越高代表今天的量能在歷史上越突出。本表篩選門檻為 75%。

**量能加速度**
最近3天的週轉率，相較於再前3天（第4至第6天）是否在放大。數值 > 1 代表量能在擴張，< 1 代表量縮。本表篩選門檻為 0.75。

**放量位置**
現價相對近3年（約756個交易日）最高價的比例，反映這波放量是在歷史相對低位還是高位發生。

**月內位置**
現價在近1個月（約21個交易日）高低區間內的相對位置，反映個股近期的強弱趨勢。

**加速趨勢（折線圖）**
連續5個時間窗口的量能加速度，最新值顯示在折線右端。可觀察量能是持續擴張、趨於穩定還是開始縮退。

**核心個股（★）**
本週上榜天數 ≥2 天的個股。在本週有效統計天數 ≤2 天時，只要出現過就算核心。

**族群評分**
綜合族群本週出現天數、每次出現的平均個股數量、平均熱度三者計算的排名依據，不直接對應任何市場數值，僅用於本表內部排序。

**巨觀族群（Macro Group）**
將細分族群歸類為更大的供應鏈方向，例如半導體供應鏈、AI/伺服器鏈等，方便從更高維度觀察資金在哪條供應鏈集中。
