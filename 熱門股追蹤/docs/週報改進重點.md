# 1. 主流延續率（最高優先）

## 目的

不要只看：

- 本週誰熱

而要看：

- 主流是否持續擴散

## 概念

例如：

IC設計：

- 第1週：2檔
- 第2週：5檔
- 第3週：9檔
- 第4週：14檔

代表：

- 資金持續擴散
- 主流仍在增強

## 建議新增欄位

```python
theme_momentum
theme_delta
breadth_expansion
```

## 建議保存格式

```text
weekly_snapshots/
    2026-05-01.json
    2026-05-08.json
    2026-05-15.json
```

## 計算方式

```python
this_week_count - last_week_count
```

或：

```python
this_week_hot - last_week_hot
```

## UI 建議

新增：

- 本週新增強勢族群
- 連續增強族群
- 熱度衰退族群

## 目的

驗證：

- 熱度是否真的有 alpha

## 必做

不要只停留在：

- 看起來合理

而要有：

- 統計證據

## 建議事件研究

### 範例

```text

之後5日平均報酬
```

```text
連續上榜 >= 3天
之後20日 alpha
```

## 建議統計

- mean return
- median return
- winrate
- sharpe
- max drawdown
- information ratio

## 建議輸出

```text
高熱度族群：
5日平均 alpha：+3.2%
勝率：61%
樣本數：1240
```

## 實作

```python
signal_day
future_return_5d
future_return_20d
```

event study。

# 6. 熱度生命周期（Lifecycle）

## 目前缺少 phase

現在只有：

- 熱不熱

但缺：

- 在哪個階段

## 建議 phase

### 1. 萌芽

- 少數股票先動
- breadth 小

### 2. 擴散

- 越來越多股票加入
- 熱度快速上升

### 3. 主升

- 高熱度
- 高成交值
- 高 breadth

### 4. 過熱

- 熱度極高
- acceleration 開始下降

### 5. 退潮

- breadth 收縮
- leader 開始轉弱

## UI

新增：

- 主流生命週期圖
